# CDMX Website Rework (v8) - Implementation Plan

> **For agentic workers:** This is a gate-driven static-site build, not a TDD code project (no test runner, pure vanilla HTML/CSS/JS + CDN libs). Each block ends with a **visual-QA step (peek)** and a **hard user gate** instead of a unit test. Do not start a block until the previous block's gate is signed off. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver two complete reworks of cdmx.be, same copy/message, visually radically different, built block-by-block under `redesign/v8/`, with the live site untouched until Renke chooses.

**Architecture:** Pure static. No bundler. Version A reuses the existing CDMX design system (dark `#0A0A0A` / sodium `#F5C800` / Hanken Grotesk) and renders Mexico City literally via a Three.js point-field of sodium lamps with nearest-neighbour connection lines (Vanta.NET style), scroll-scrubbed. Version B is a full rebrand (light, trend palette, new fonts) reusing the *same* WebGL point-field technique but **abstract** (topography/nodes/grid, no skyline). Both share one copy-deck and a strict performance/a11y contract (lazy WebGL, reduced-motion poster fallback, mobile fallback, dispose on teardown).

**Tech Stack:** HTML5, CSS custom properties, vanilla ES modules, GSAP 3.12.7 + ScrollTrigger (CDN), Lenis 1.1.18 (CDN), Three.js 0.160 via ESM importmap (CDN), Canvas2D fallback. Self-hosted fonts/logos/images reused from `../../assets/`.

**Emotional spine (both versions):** Mexico City as melting pot, old culture, startups and innovation bound together; from "murder capital" to thriving through pure connection. Metaphor for the work: connect silos and disciplines so both company and people grow and stand in life with pride.

**Hard house rules (apply to every block):**
- No em-dashes anywhere in copy (use periods, commas, or restructure). A pre-commit hook blocks them.
- Exact verified numbers only: `EUR 2M+`, `production AI (no pilots)`, `18 months embedded`, `8+ years`, `3 postgraduate degrees`, `30+ person agency`, `branch manager at KIXX from July 1, 2026`. Never round, never inflate.
- KIXX is **proof, not hero.** The previous v7-boardroom "external CEO" framing is deliberately dropped. KIXX appears as block 7 only: factual, calm, innovation lens.
- Tone: position-taking, serving not pleasing, running prose (renke-voice). No emoji, no exclamation marks, no "book a free call".
- Accessibility and Core Web Vitals are acceptance criteria, not nice-to-haves.

## Task A.1: Signature visual - sodium-lamp point-field + connections

> This is the riskiest, highest-value piece. Build and prove it in isolation before any block copy. It is reused (re-skinned) by Version B.

**Files:**
- Create: `redesign/v8/version-a-mexico/js/point-field.js`
- Create: `redesign/v8/version-a-mexico/js/scene-a.js`

- [ ] **Step 1: Invoke specialist-3d**

Invoke `specialist-3d`. Brief: THREE.Points field of sodium-yellow lamps over a city-like distribution, additive blending, soft radial glow in fragment shader, per-point brightness animated from 0 ("emerging") scrubbed on scroll; nearest-neighbour connection lines (each lamp to N closest neighbours, line alpha by distance) drawn in via per-segment alpha on scroll; `UnrealBloomPass` desktop/high-end only; lazy-init via IntersectionObserver; pause render loop offscreen and on hidden tab; `setPixelRatio(min(dpr, 2))`; dispose on teardown.

- [ ] **Step 2: Write `point-field.js` (the reusable module)**

```js
// redesign/v8/version-a-mexico/js/point-field.js
import * as THREE from 'three';
import { isLowEnd, clamp } from './lib.js';

// config lets Version B reuse this with abstract distribution + palette.
export function createPointField(canvas, config = {}) {
  const {
    count = isLowEnd() ? 1200 : 3500,
    color = 0xF5C800,
    neighbours = 3,
    bloom = !isLowEnd(),
    distribute = defaultCityDistribution, // (i, n) => {x,y,z}
  } = config;

  const renderer = new THREE.WebGLRenderer({ canvas, antialias: true, alpha: true });
  renderer.setPixelRatio(Math.min(window.devicePixelRatio || 1, 2));
  renderer.setSize(window.innerWidth, window.innerHeight, false);

  const scene = new THREE.Scene();
  const camera = new THREE.PerspectiveCamera(55, window.innerWidth / window.innerHeight, 0.1, 100);
  camera.position.set(0, 0, 14);

  // --- points ---
  const positions = new Float32Array(count * 3);
  const brightness = new Float32Array(count); // 0..1, animated by scroll
  for (let i = 0; i < count; i++) {
    const p = distribute(i, count);
    positions[i * 3] = p.x; positions[i * 3 + 1] = p.y; positions[i * 3 + 2] = p.z;
    brightness[i] = 0;
  }
  const geo = new THREE.BufferGeometry();
  geo.setAttribute('position', new THREE.BufferAttribute(positions, 3));
  geo.setAttribute('aBright', new THREE.BufferAttribute(brightness, 1));

  const mat = new THREE.ShaderMaterial({
    transparent: true, depthWrite: false, blending: THREE.AdditiveBlending,
    uniforms: { uColor: { value: new THREE.Color(color) }, uSize: { value: 22.0 } },
    vertexShader: `
      attribute float aBright; varying float vB;
      uniform float uSize;
      void main(){ vB = aBright;
        vec4 mv = modelViewMatrix * vec4(position,1.0);
        gl_PointSize = uSize * (1.0 + aBright) * (1.0 / -mv.z);
        gl_Position = projectionMatrix * mv; }`,
    fragmentShader: `
      varying float vB; uniform vec3 uColor;
      void main(){ vec2 c = gl_PointCoord - 0.5;
        float d = length(c);
        float glow = smoothstep(0.5, 0.0, d);   // soft radial glow
        gl_FragColor = vec4(uColor, glow * vB); }`,
  });
  const points = new THREE.Points(geo, mat);
  scene.add(points);

  // --- nearest-neighbour connections ---
  const segs = buildConnections(positions, count, neighbours); // [{a,b,dist}]
  const linePos = new Float32Array(segs.length * 6);
  const lineAlpha = new Float32Array(segs.length * 2);
  segs.forEach((s, k) => {
    linePos.set([positions[s.a*3],positions[s.a*3+1],positions[s.a*3+2],
                 positions[s.b*3],positions[s.b*3+1],positions[s.b*3+2]], k*6);
  });
  const lgeo = new THREE.BufferGeometry();
  lgeo.setAttribute('position', new THREE.BufferAttribute(linePos, 3));
  lgeo.setAttribute('aAlpha', new THREE.BufferAttribute(lineAlpha, 1));
  const lmat = new THREE.ShaderMaterial({
    transparent: true, depthWrite: false, blending: THREE.AdditiveBlending,
    uniforms: { uColor: { value: new THREE.Color(color) } },
    vertexShader: `attribute float aAlpha; varying float vA;
      void main(){ vA = aAlpha; gl_Position = projectionMatrix * modelViewMatrix * vec4(position,1.0);} `,
    fragmentShader: `varying float vA; uniform vec3 uColor;
      void main(){ gl_FragColor = vec4(uColor, vA * 0.5); }`,
  });
  const lines = new THREE.LineSegments(lgeo, lmat);
  scene.add(lines);

  // --- optional bloom (desktop/high-end only) ---
  let composer = null;
  if (bloom) {
    // dynamic import keeps addons off the critical path for low-end devices
    Promise.all([
      import('three/addons/postprocessing/EffectComposer.js'),
      import('three/addons/postprocessing/RenderPass.js'),
      import('three/addons/postprocessing/UnrealBloomPass.js'),
    ]).then(([{ EffectComposer }, { RenderPass }, { UnrealBloomPass }]) => {
      composer = new EffectComposer(renderer);
      composer.addPass(new RenderPass(scene, camera));
      composer.addPass(new UnrealBloomPass(
        new THREE.Vector2(window.innerWidth, window.innerHeight), 0.8, 0.6, 0.85));
    });
  }

  // --- scroll-driven reveal API ---
  let progress = 0; // 0..1 set by ScrollTrigger
  function setProgress(p) {
    progress = clamp(p, 0, 1);
    const b = geo.attributes.aBright.array;
    for (let i = 0; i < count; i++) {
      const threshold = i / count;               // points light up in order
      b[i] = clamp((progress - threshold) * 6, 0, 1);
    }
    geo.attributes.aBright.needsUpdate = true;
    const la = lgeo.attributes.aAlpha.array;
    for (let k = 0; k < segs.length; k++) {
      const t = k / segs.length;
      const a = clamp((progress - t) * 4, 0, 1) * (1 - clamp(segs[k].dist / 4, 0, 1));
      la[k*2] = a; la[k*2+1] = a;
    }
    lgeo.attributes.aAlpha.needsUpdate = true;
  }

  // --- render loop with offscreen / hidden-tab pause ---
  let running = false, raf = 0;
  function frame() {
    if (!running) return;
    points.rotation.y += 0.0006;
    lines.rotation.y = points.rotation.y;
    (composer || renderer).render(scene, camera);
    raf = requestAnimationFrame(frame);
  }
  function start() { if (!running && !document.hidden) { running = true; frame(); } }
  function stop() { running = false; cancelAnimationFrame(raf); }
  document.addEventListener('visibilitychange', () => document.hidden ? stop() : start());

  function resize() {
    renderer.setSize(window.innerWidth, window.innerHeight, false);
    camera.aspect = window.innerWidth / window.innerHeight; camera.updateProjectionMatrix();
    composer?.setSize(window.innerWidth, window.innerHeight);
  }
  window.addEventListener('resize', resize);

  function dispose() {
    stop(); window.removeEventListener('resize', resize);
    geo.dispose(); mat.dispose(); lgeo.dispose(); lmat.dispose();
    composer?.dispose?.(); renderer.dispose();
  }

  return { start, stop, setProgress, dispose };
}

function defaultCityDistribution(i, n) {
  // grid-ish ground plane with vertical "towers" -> reads as a city at distance
  const cols = Math.ceil(Math.sqrt(n));
  const gx = (i % cols) / cols - 0.5, gz = Math.floor(i / cols) / cols - 0.5;
  const r = Math.hypot(gx, gz);
  const h = Math.max(0, (0.5 - r)) * (Math.sin(i * 12.9898) * 0.5 + 0.5) * 6;
  return { x: gx * 26, y: h - 2, z: gz * 26 };
}

function buildConnections(pos, n, k) {
  const segs = [];
  for (let i = 0; i < n; i++) {
    const dists = [];
    for (let j = 0; j < n; j++) {
      if (i === j) continue;
      const dx = pos[i*3]-pos[j*3], dy = pos[i*3+1]-pos[j*3+1], dz = pos[i*3+2]-pos[j*3+2];
      dists.push({ j, d: dx*dx+dy*dy+dz*dz });
    }
    dists.sort((a, b) => a.d - b.d);
    for (let m = 0; m < k; m++) {
      if (i < dists[m].j) segs.push({ a: i, b: dists[m].j, dist: Math.sqrt(dists[m].d) });
    }
  }
  return segs;
}
```

> Note: the O(n squared) `buildConnections` runs once at init. For `count > 4000` replace with a spatial grid; default counts (1200/3500) are fine. Document this in a comment.

// abstract: a flowing contour surface, no skyline
function contourDistribution(i, n) {
  const cols = Math.ceil(Math.sqrt(n));
  const gx = (i % cols) / cols - 0.5, gz = Math.floor(i / cols) / cols - 0.5;
  const y = Math.sin(gx * 8) * Math.cos(gz * 8) * 2.2; // smooth topography

}

export function createAbstractField(canvas) {

    neighbours: 3,
    distribute: contourDistribution,
  });
}
